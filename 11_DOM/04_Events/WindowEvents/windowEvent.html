Window Events

Window events are events that occur on the global window object in a web browser. These events can be triggered by user interactions, such as resizing the browser window, scrolling the page, or loading a new page. You can listen for these events using JavaScript to perform actions in response to them.

Some common window events include:

  • load: Triggered when the entire page has loaded.
  • resize: Triggered when the browser window is resized.
  • scroll: Triggered when the user scrolls the page.
  • beforeunload: Triggered before the user leaves the page.
