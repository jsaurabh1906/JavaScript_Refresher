Manipulating the DOM

In this Example, we will manipulate the DOM by changing the content of elements.

Original Text

This is a paragraph.

After using textContent :

            
                paragraph.textContent = "The text has been changed!";

Result:

The text has been changed!

After using innerHTML

        
            paragraph.innerHTML = "The text has been changed! <strong>This is bold.</strong>"

Result: The text has been changed! This is bold.

After using innerText

        
            paragraph.innerText = "The text has been changed! This will not be bold."

The text has been changed! <strong>This will not be bold.</strong>
